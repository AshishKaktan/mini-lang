§Introduction

mini-lang is overlay programming language build on top of python, with aim to provide extra flavor to primary language.

§Quick Start

Currently mini-lang is at it's alpha stage so we are not having any e2e (end-to-end) running module yet. So, it's a base skeleton for upcoming flavors to this programming language.

§Requirements

  1. python v3.0 or greater

§Download & Run

git clone https://github.com/hackerfrog/mini-lang.git

python shell.py

§Exit shell.py

To exit from infinite loop of shell, type :x or ::exit.
